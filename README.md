## 一、项目概述

本项目是我在**北京智谱华章科技股份有限公司**参与的GLM-4.5V 多模态大模型"图像 - SVG 生成调优"项目。

项目聚焦于多模态大模型的图像理解与代码生成能力优化，针对**图像到 SVG 可缩放矢量图形代码**的智能生成进行深入研究与调优。通过指令微调、Prompt 工程优化和多Agent校验工作流设计，显著提升了模型对复杂图形结构的语义理解精度与SVG代码生成正确率。

![img](https://cdn.nlark.com/yuque/0/2026/png/23187920/1775999287353-557ea3f9-753b-42f1-a512-d6de334ba9ef.png)

## 二、技术栈

- **核心框架**：Python 3.10+ / PyTorch 2.0+
- **多模态大模型**：GLM-4.5V（智谱自研）
- **模型微调**：LoRA (Low-Rank Adaptation)、DeepSpeed
- **提示词工程**：Chain-of-Thought、Few-Shot Learning
- **评估框架**：BLEU、CLIP Score、语义相似度比对
- **Agent工作流**：Multi-Agent Orchestration、LangChain
- **部署优化**：TensorRT、模型量化、vLLM推理加速

## 四、项目技术细节

### 4.1 图像到SVG转换技术框架

```plain
输入图像 → 视觉编码器 → 语义特征提取 → 跨模态对齐
                                         ↓
                                   LLM解码器 → SVG代码生成
                                         ↓
                              校验Agent → 语法校验 / 渲染验证
                                         ↓
                                    最终输出
```

### 4.2 LoRA微调策略

| **阶段** | **目标**             | **关键技术**   |
| -------- | -------------------- | -------------- |
| 基础微调 | 建立图像-SVG映射能力 | LoRA-Rank 8/16 |
| 语义增强 | 提升复杂图形理解     | 课程学习策略   |
| 细节优化 | 精确控制生成质量     | 差异化学习率   |

### 4.3 校验Agent工作流

```plain
生成SVG → Agent-1(语法) → Agent-2(语义) → Agent-3(视觉)
              ↓               ↓               ↓
          错误修正 ← 多Agent协商 → 质量评分
                            ↓
                      最终判定(通过/修正/重生成)
```
